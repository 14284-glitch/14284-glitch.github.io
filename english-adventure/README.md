# English Adventure 英語生活大冒險

這是適合國小兒童、優先支援平板橫向操作的純前端英文學習遊戲。

## 本機開啟

直接使用 Chrome、Edge 或 Safari 開啟 `index.html`。語音辨識與部分麥克風功能在 HTTPS 網站上最穩定。

## GitHub Pages

將 `english-adventure` 資料夾連同主網站上傳到 GitHub Pages 儲存庫，即可由 `/english-adventure/` 路徑開啟。

## 第一階段

- 六個生活情境與 60 張卡牌
- 卡牌翻面、收藏、正常／慢速發音、例句與對話播放
- 看圖選句子、聽聲選圖片、句子排序
- 星星、經驗值、等級、解鎖、徽章與 localStorage 紀錄
- 家長／教師學習紀錄與清除確認
- 平板、手機、電腦響應式與減少動態效果
